ML Model Comparison 

Approach
The following steps were performed to complete this assignment:
Loaded the Iris dataset using sklearn.datasets.
Split the dataset into training and testing sets (80% training, 20% testing).
Initialized three different models:
Logistic Regression
K-Nearest Neighbors (KNN)
Decision Tree
Trained each model on the training dataset.
Made predictions using the test dataset.
Evaluated the performance of each model using accuracy score.
Compared the results to determine the best-performing model


Difficulties Faced
Selecting appropriate parameters for models like KNN (e.g., choosing the value of K)
Observing minimal differences in accuracy between models, making the comparison less obvious

Solutions
Used default parameters to maintain simplicity
Focused on understanding model behaviour rather than only accuracy
Compared models based on simplicity, speed, and interpretability


 Learnings
Different models can perform equally well on simple datasets
Model selection depends on dataset complexity and use case
Simpler models like Logistic Regression can often be very effective
Understanding the working of models is as important as their accuracy
